# LST_stratified_Taylor_Green_vortex_uvp_matrix_free
This is the matrix-free linear stability analysis code for flow with 2D shear flows. This code will aim to reprodue results in Guo J, Taylor JR, Zhou Q. Zigzag instability of columnar Taylor–Green vortices in a strongly stratified fluid. Journal of Fluid Mechanics. 2024 Oct;997:A34.
